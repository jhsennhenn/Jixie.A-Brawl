# What is this project?
    This is a mod for the first-person shooter Titanfall 2 that edits the ability "A-Wall" 
    to follow the player and act as a personal shield to cover the front half of the player from damage.
 
# TO RUN AND DOWNLOAD
    * Background:
     You must have have the Northstar Client downloaded, as well as Titanfall 2 on Steam or EA.
     _If you do not already have Northstar downloaded, you can follow the instructions on 
     [The Northstar Website](https://northstar.tf/)._
     
    1. Download this mod on a Northstar Mod Manager like [Thunderstore](https://thunderstore.io/c/northstar/), 
       or manually by downloading the zip file in the "<> code" section or assets linked below.
    
    2. Navigate to the "Jixie.A_Brawl" folder that contains the "mod" folder and "mod.json" file.
    
    3. Open a second window in file explorer in your Titanfall 2 game files. On Steam, this can be done by 
       right-clicking on the game in your library, and clicking "**browse**" in "**installed files**".
    
    4. Open this second window at "Titanfall2/R2Northstar/mods" and drop the "Jixie.A_Brawl" folder into the 
       mods folder.
    
    5. Now, run your game and make sure the mod is activated on the Mods page on the title screen. 
       Just equip A-Wall as your ability in-game make sure your server supports this mod.
